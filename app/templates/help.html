Help
Getting Started
• Create an account (top bar)
• Upload a video (UPLOAD tab)
• Watch + Like + Favorite videos
Accounts
• Username: 3-20 (a-z, 0-9, _)
• Password: min 6 chars
• Keep your password safe
Videos
• Formats: mp4, webm, ogg, mov, mkv
• Thumbnails: png/jpg/webp (optional)
• If no thumbnail: auto-generate (needs ffmpeg)
Favorites
• Open a video page
• Click "Add/Remove Favorite"
• View them in FAVORITES tab
Messages
• Go to MESSAGES tab
• Compose → send to a username
• Inbox/Sent available
Troubleshooting
• Upload error? Check file size limit
• No auto-thumbnail? Install ffmpeg
• Video not loading? Try mp4 format
